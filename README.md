# inventario-web-tfg
Repositorio para la Aplicación Web SPA con Angular + Spring Boot + MySQL de control de inventarios "StockFlow" para el TFG del ciclo de DAW en Ilerna Online

Descripción General

Este proyecto consiste en el desarrollo de una aplicación web de control de inventarios, diseñada como Trabajo de Fin de Grado Superior.
El objetivo principal es permitir a pequeñas y medianas empresas gestionar su inventario de forma eficiente mediante una interfaz moderna y un backend robusto.

El sistema permite:

Registro y autenticación de usuarios

Gestión de productos (CRUD)

Registro de movimientos de inventario (entradas y salidas)

Control automático de niveles de stock

Listados filtrables con búsqueda

Dashboard general del inventario

Comunicación completa mediante API REST

La aplicación se ha desarrollado siguiendo buenas prácticas de arquitectura, seguridad, documentación y mantenimiento.
