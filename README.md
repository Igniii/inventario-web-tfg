# inventario-web-tfg
Repositorio para la Aplicación Web SPA con Angular + Spring Boot + MySQL de control de inventarios "StockFlow" para el TFG del ciclo de DAW en Ilerna Online

Descripción General

Este proyecto consiste en el desarrollo de una aplicación web de control de inventarios, diseñada como Trabajo de Fin de Grado Superior.
El objetivo principal es permitir a pequeñas y medianas empresas gestionar su inventario de forma eficiente mediante una interfaz moderna y un backend robusto.

El sistema permite:

Registro y autenticación de usuarios
Gestión de productos (CRUD)
Registro de movimientos de inventario (entradas y salidas)
Control automático de niveles de stock
Listados filtrables con búsqueda
Dashboard general del inventario
Comunicación completa mediante API REST
La aplicación se ha desarrollado siguiendo buenas prácticas de arquitectura, seguridad, documentación y mantenimiento.

Objetivos: 
Implementar un sistema de autenticación a la plataforma para poder hacer uso de esta.
Crear un módulo de productos y componentes con operaciones CRUD (crear, leer, actualizar, eliminar) para mantener el inventario actualizado.
Diseñar un módulo de categorías y proveedores, permitiendo organizar los elementos del inventario de manera estructurada y asociar proveedores a cada producto.
Gestionar los movimientos de entrada y salida de stock, registrando fechas, cantidades y responsables de cada operación.
Implementar alertas automáticas de stock mínimo para productos críticos, facilitando la planificación de reposiciones.
Desarrollar un panel de control con informes y gráficos que muestre estadísticas del inventario.
Crear una interfaz web responsive e intuitiva usando Angular, HTML y CSS, accesible desde cualquier dispositivo.
Garantizar la integración y comunicación entre frontend y backend mediante APIs REST desarrolladas en Java con Spring Boot y gestión de base de datos en MySQL.
